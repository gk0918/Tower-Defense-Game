üè∞ TOWER DEFENCE GAME:

Tower Defense Game is a 2D strategy game built using Python and Pygame.
The player places defensive towers along a path to stop waves of enemies from reaching the base. This project demonstrates game logic, path movement, enemy waves, tower attacks, scoring, and resource management in a clear and easy-to-understand way.

üéÆ GAME OVERVIEW:

Enemies follow a fixed path toward your base.
You place towers at strategic positions to stop them.
Towers automatically detect and shoot enemies in range.
Each enemy destroyed gives you points/money.
Use resources wisely to upgrade or place more towers.
Game difficulty increases with each wave.
